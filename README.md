# 📄 AI Resume Analyzer Dashboard

A full-stack web application that analyzes resumes, extracts skills, and provides a scoring system with an interactive dashboard.

## 🚀 Tech Stack
- Frontend: Streamlit
- Backend: Flask
- Database: MySQL

## 🌍 Features
- Analyze resume text
- Extract skills dynamically
- Generate resume score
- Store results in MySQL
- Display analytics dashboard

## 🏗️ Architecture
Streamlit → Flask API → MySQL Database

## ▶️ How to Run

### 1. Backend
cd backend
python app.py

### 2. Frontend
cd frontend
python -m streamlit run app.py
